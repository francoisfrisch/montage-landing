Ninja is an advanced web application built in HTML5. It targets creative professionals who want to create HTML5 content for desktop and mobile deployment. Supporting technologies such as SVG, Canvas, and WebGL. You can create or import expressive creative content and combine this with HTML5 elements and Montage components all within Ninja.

Screening is a web-based test automation tool you can use to create, store and execute test cases and view test results. You can use Screening to automate UI test cases for web-based applications using recording, scripting, and debugging functionality.
