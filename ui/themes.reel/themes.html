Themes

Montage comes with the Skeleton theme. It's a basic, neutral-looking theme for HTML5 controls and inputs. It can be used as wireframe theme for prototyping, easily be styled with CSS to create custom Skins or it can be used as a starting point for creating your own themes. It also aims to provide cross-browser support so it can serve as a fallback theme.
